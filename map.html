Brand in one breath

Color as accent

Orange = action, periwinkle = cool accent, ink = text. White space is the canvas — never wallpaper. One gradient element per view.

Expressive + functional type

Clash Display for headlines & KPI values; Satoshi for body, UI, tables; Bricolage for editorial marketing accents.

Soft & diffuse

Soft corners (12 / 24 / 32px), soft vertical shadows, a warm orange glow on the primary button hover. Nothing sharp.

Rules
  • Radii: inputs/buttons 12px, cards 24px, large surfaces 32px, pills round.
  • Shadows: soft, vertical, low-contrast. Primary button gets a warm orange glow on hover.
  • Light app vs dark site — only surfaces/text invert; accents stay constant.
  • Motion: gentle ease-out, 120–360ms. No bounce or infinite decoration.

How to use

Installed as the skill alfred-ai-design (user + project level). Invoke /alfred-ai-design and design on-brand.

Directly

Link styles.css, load _ds_bundle.js, destructure components. data-theme="dark" for dark.

Voice & icons

Voice
  • First-person chief of staff: Alfred says "I", speaks to "you".
  • Lead with the decision or the number, then the reasoning.
  • Sentence case everywhere; UPPERCASE only for small tracked eyebrows.
  • No emoji. Warmth comes from palette and type, not decoration.
Icons & logos
  • Custom single-color SVG icon set (line style, ~16–24px), tinted via currentColor.
  • Never substitute emoji or unicode symbols.
  • Logo lockups in color + all-white; white only on ink or the gradient. Never recolor the gradient mark.
