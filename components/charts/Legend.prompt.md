# Legend

Swatch + label key for any chart. Row (default) or column layout; each item may carry an optional right-aligned value. Colors default through the brand categorical palette so a legend always matches the chart beside it.

## Props

_Props are documented in the `.d.ts`._

## Usage

```jsx
const { Legend } = window.AlfredAIDesignSystem_1ce241;

<Legend />
```

## Notes
- Styled entirely from design-system tokens (`var(--…)`); it inherits the active theme, so it works on the light app and the dark website without changes.
- Sentence case, first-person "chief of staff" voice in copy. No emoji.
